# Adeptus Mechanicus Theme

> **Warhammer 40K Adeptus Mechanicus themes - Dark forge world and light parchment themes from The Grimdarkness of the Far future. Praise the Omnissiah\!**


<br>
This repository contains the **Adeptus Mechanicus Theme** for IntelliJ IDEA and other JetBrains IDEs. It provides a visual experience inspired by the Tech-Priests of Mars or the "Cult Mechanicus" from Warhammer 40000.
<br><br>

## ‚ú® Features

The plugin includes two distinct themes, each featuring distinct **Dark** and **Light** color schemes to honor the Machine God.

### 1\. Adeptus Mechanicus Dark (Forge World)

This dark theme is inspired by the deepest, crimson-lit forges of a Forge World, emphasizing contrast and readability for long coding sessions.
Use of this theme is advised to avoid overloading your optical implants / cogitator.

| UI Element | Color | Hex Code | Purpose/Aesthetic |
| :--- | :--- | :--- | :--- |
| **Primary Background** | Deep Crimson/Black | `#1A0A0A` | Represents the forge world's core. |
| **Primary Foreground** | Gold/Brass | `#D4AF37` | Standard text and primary focus elements. |
| **Selection Background** | Dark Red/Maroon | `#8B0000` | Highlighting selected text/elements. |
| **Caret/Numbers** | Bright Gold | `#FFD700` | Indicates active elements and numerical values. |

<br>

### 2\. Adeptus Mechanicus Light (Sacred Data-Slate)

This light theme mimics the look of ancient parchment and data-slates, using warm, sepia tones.

| UI Element | Color | Hex Code | Purpose/Aesthetic |
| :--- | :--- | :--- | :--- |
| **Primary Background** | Warm Parchment/Beige | `#F5F0E6` | The base color, like a digital scroll. |
| **Primary Foreground** | Dark Brown/Sepia | `#4A3A2A` | Standard text color, resembling ink. |
| **Selection Background** | Gold/Brass | `#D4AF37` | Highlighting selected text/elements. |
| **Keywords/Tags** | Maroon/Red | `#8B0000` | Reserved for language keywords and markup tags. |

## üíª Installation

There are two primary methods to install the **Adeptus Mechanicus Theme** for your JetBrains IDE:

### 1\. From the JetBrains Plugin Marketplace (Recommended)

This is the easiest and most recommended way to install the theme.

1. ¬†Open **Settings/Preferences** in your JetBrains IDE.
2. ¬†Navigate to **Plugins** > **Marketplace** tab.
3. ¬†Search for **"Adeptus Mechanicus Theme"**.
4. ¬†Click the **Install** button.
5. ¬†Restart the IDE when prompted to complete the installation.
6. ¬†Select the theme 'Mechanicus Forge World'(dark) or 'Mechanicus Sacred Data-Slate'(light) from the Appearance settings.
7. ¬†Enojoy!

<hr>

### 2\. Manual Installation (From Disk)

If you have downloaded the plugin `.jar` file directly, you can install it manually:

1. Download the `.jar` file from [Releases](https://github.com/Chiarchiaooo/intellij_admech40k_theme/releases/latest).
2. ¬†Open **Settings/Preferences** in your JetBrains IDE.
3. ¬†Navigate to **Plugins**.
4. ¬†Click the **‚öôÔ∏è** icon (or the gear icon) and select **"Install Plugin from Disk..."**.
5. ¬†Select this downloaded `.jar` file.
6. ¬†Restart the IDE when prompted.

<br>

### Applying the Theme

After installation, you can switch between the dark and light themes:
1.  Go to **Settings/Preferences** -\> **Appearance & Behavior** -\> **Appearance**.
2.  In the **Theme** dropdown, select either **'Mechanicus Forge World'(dark)** or **'Mechanicus Sacred Data-Slate'(light)**.
3.  Enjoy!
